# Design Style: Corporate Trust

## 1. Design Philosophy
This style embodies the **modern enterprise SaaS aesthetic** — professional yet approachable, sophisticated yet friendly. 

**Core Principles:**
- **Trustworthy Yet Vibrant**: Establishes credibility through clean structure and professional typography while maintaining visual energy through vibrant gradients and colorful accents.
- **Dimensional Depth**: Uses isometric perspectives, soft colored shadows, and subtle 3D transforms.
- **Refined Elegance**: Every element is polished with attention to micro-interactions and smooth transitions.
- **Purposeful Gradients**: Indigo-to-violet gradients serve as the visual signature.

## 2. Design Token System (Update CSS variables with these)

### Colors (Light Mode)
*   **Background**: `#F8FAFC` (Slate 50) - A very subtle cool grey/white base.
*   **Foreground (Surface/Cards)**: `#FFFFFF` (White) - For cards and raised elements.
*   **Primary**: `#4F46E5` (Indigo 600) - The core brand color. Vibrant blue-purple.
*   **Secondary**: `#7C3AED` (Violet 600) - For gradients and accents.
*   **Text Main**: `#0F172A` (Slate 900) - High contrast, sharp.
*   **Text Muted**: `#64748B` (Slate 500) - For supporting text.
*   **Accent/Success (Gainers)**: `#10B981` (Emerald 500) - For positive indicators.
*   **Negative (Losers)**: `#EF4444` (Red 500) - For negative indicators.
*   **Border**: `#E2E8F0` (Slate 200) - Subtle separation.

### Typography
*   **Font Family**: `Plus Jakarta Sans` — Geometric sans-serif with friendly rounded terminals.

### Radius & Border
*   **Radius**: `12px` (`rounded-xl`) for cards, `8px` (`rounded-lg`) for inputs. Buttons are `rounded-full` or `rounded-lg`.
*   **Borders**: Thin, 1px borders using `#E2E8F0`.

### Shadows & Effects (Use Colored Shadows)
*   **Default Card Shadow**: `0 4px 20px -2px rgba(79, 70, 229, 0.1)` — Soft blue-tinted base elevation.
*   **Hover Card Shadow**: `0 10px 25px -5px rgba(79, 70, 229, 0.15), 0 8px 10px -6px rgba(79, 70, 229, 0.1)`
*   **Button Shadow**: `0 4px 14px 0 rgba(79, 70, 229, 0.3)`
*   **Primary Gradient**: `linear-gradient(to right, #4F46E5, #7C3AED)`

## 3. Component Stylings

### Buttons & Badges
*   **Primary Action / Selected State**: Gradient background (Indigo to Violet). White text. Button Shadow applied. Transition: Lift (`-translate-y-0.5`) and increase shadow on hover.
*   **Badges**: Icon/Text in `text-indigo-600` on `bg-indigo-50` container.

### Cards & Panels
*   **Base**: White background (`#FFFFFF`), `12px` border radius, `1px solid #E2E8F0` border, Default Card Shadow.
*   **Hover/Active**: Slight lift (`transform: translateY(-4px)`) and increased shadow intensity.

### Interactive Charts & Tables
*   **Tables**: Clean layout with `#E2E8F0` borders separating rows.
*   **Charts**: Update chart lines/fills to utilize the Indigo and Violet core colors, alongside Emerald for positive trends and Red for negative trends.

## 4. Non-Generic Bold Choices (Implement where applicable)
*   **Hover Transforms**: Apply subtle 3D movement on interaction for cards (e.g., `hover:rotate-x-[2deg] hover:rotate-y-[-8deg]`).
*   **Gradients**: Use the primary Indigo-to-Violet gradient on primary metrics, active sidebar links, or the main call-to-action buttons.
*   **Micro-Interactions**: Smooth `transition-all duration-200 ease-out` on all interactive elements.
