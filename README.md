# 📈 Stock Market Analyzer

A production-grade, modular Python web application built with Streamlit that analyzes historical price action, calculates technical indicators, and leverages an AI (PyTorch LSTM) model to predict near-term stock market movements. 

The application features a modern, clean financial aesthetic with an intuitive navigation sidebar, detailed interactive Plotly charts, and an automated S&P 500 stock screener.

## ✨ Key Features

1. **📊 Market Overview**
   - Fetch high-frequency historical and live market data (OHLCV) using `yfinance`.
   - Retrieve corporate financial metrics (Revenue, Net Income, EPS) via the SEC EDGAR API.
   - Interactive candlestick and volume charts.

2. **🔬 Technical Analysis**
   - Extracts and visualizes standard trading indicators: Moving Averages (20/50-day SMA & EMA), RSI, MACD, and Bollinger Bands.
   - Computes statistical signals like rolling volatility and momentum.

3. **🤖 AI Prediction**
   - Utilizes a PyTorch LSTM (Long Short-Term Memory) neural network.
   - Processes sequences of 19 technical and price features to predict the probability of a stock's price going UP or DOWN over a configurable horizon.
   - Radial gauges and confidence metrics for clear interpretation of the model's signals.

4. **📈 Backtest Results**
   - Simulates trading the AI's signals against historical data.
   - Calculates real-world portfolio performance metrics: Total Return, Max Drawdown, Sharpe Ratio, and Win Rate.
   - Compares the AI strategy against a standard "Buy & Hold" baseline.

5. **🏆 Stock Screener**
   - Automatically scans the S&P 500 companies.
   - Ranks stocks using a composite technical score based on moving averages, momentum, and oversold conditions.
   - Presents top picks with rich visual cards and sector-based histograms.

6. **🛠 Admin Data**
   - Persistent, dedicated logging tab tracking data pipeline operations, model training loss, and system diagnostics without cluttering the main UI.

## 🛠 Technology Stack

* **Frontend / UI:** Streamlit, Vanilla CSS injections for a premium light-theme aesthetic.
* **Data Visualizations:** Plotly (Candlestick, Area, Spline, Histogram, Radial Gauges).
* **Machine Learning:** PyTorch (LSTM), Scikit-Learn (Feature Scaling).
* **Data Processing:** Pandas, NumPy.
* **Data Sourcing:** `yfinance`, SEC EDGAR API.

## 🚀 Installation

1. **Clone the repository** (or navigate to the project directory):
   ```bash
   cd stock_analyzer_project
   ```

2. **Create a virtual environment** (recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate
   ```

3. **Install the dependencies**:
   Make sure you have the required packages installed. 
   ```bash
   pip install streamlit pandas numpy torch scikit-learn yfinance plotly python-dotenv requests
   ```

## 🎮 Usage

Run the Streamlit application from your terminal:

```bash
streamlit run app.py
```

1. Open the provided `localhost` URL in your web browser.
2. Use the **Navigation** sidebar to toggle between the **Analyzer**, **Screener**, and **Admin Data** modes.
3. In the Analyzer, input a Ticker (e.g., AAPL), set your date ranges and model parameters, and click **Run Analysis**.
4. Navigate through the tabs to explore technicals, AI predictions, and backtest results!
